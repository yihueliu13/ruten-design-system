Token Architecture
comp/button/
├── {role}/               ← primary / secondary / tertiary / ghost
│   ├── {status}/         ← default / danger / info
│   │   └── {state}/     ← default / hover / pressed / disabled / loading
│   │       └── {prop}    background, text-color, border-color, opacity
│   └── disabled/border-width ← tertiary only: 0.5px
├── {size}/               ← xs / sm / md / lg / xl
│   └── {prop}            min-height, padding-h, padding-v, font-size, icon-size, gap, border-radius
├── base/font-weight       ← 500 (shared)
└── form                  ← label / label-icon / icon (Figma variant prop)
